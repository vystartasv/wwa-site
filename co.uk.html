Agent-native

For AI agents

All twelve checks are MCP tools: typed inputs, JSON findings with clause references. Point any MCP client at the server and call bidcheck(), pricecheck() and the rest inside your own workflows.

Run the lightweight proxy, or fully offline with your own key — no domain dependency, nothing leaves your infrastructure.

View on GitHub →  ·  Developer docs →

# Proxy mode (hosted API)
docker run -p 3100:3100 ghcr.io/vystartasv/agent-tools:mcp

# Direct mode (offline, your key)
docker run -e GROQ_API_KEY=*** -p 3101:3101 \
  ghcr.io/vystartasv/agent-tools:mcp-direct

# Any MCP client
GET  /mcp/tools        → list all 12 tools
POST /mcp/tools/call   → invoke a tool
