MCP Server

Works With Agents — MCP Server

MCP server with 12 tools for UK tender bid compliance, market scanning, and tender discovery. Open-source, self-hostable. Python, TypeScript, Go, Rust SDKs.

Try the tools at workswithagents.com View on GitHub

Quick start

# Proxy mode (hosted API)
docker run -p 3100:3100 ghcr.io/vystartasv/agent-tools:mcp

# Direct mode (offline, your key)
docker run -e GROQ_API_KEY=*** -p 3101:3101 ghcr.io/vystartasv/agent-tools:mcp-direct

# Any MCP client
GET  /mcp/tools        list all 12 tools
POST /mcp/tools/call   invoke a tool
